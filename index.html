Media Queries

Media queries help make a website responsive and compatible with all different types of devices. Using @media screen and (max-width: 768px) would make the following CSS attributes apply when the screen resolution goes below 768px. The following sinppet changes the box layout to show 2 at a time on screens below 768px.

@media screen and (max-width: 768px) {
.box-1 {
width: 48%;
}
}

Flex Wrap

Use flex-wrap to make the container's children flexible. The contents in the container will resize and fit to the viewport as necessary. This code snippet is what makes each of these boxes felxible.

.all-boxes> {
display: flex;
flex-wrap: wrap;
margin-top: 2%;
justify-content: space-around;
}

Position Absolute

Use position: absolute combined with position: relative in the relative item to move the desired item where you want. This code snippet made the header of this box centered on the dotted line using negative top.

.box-1 h2 {
border: 3px solid blanchedalmond;
position: absolute;
top:-25px;
width: 80%;
text-align: center;
left: 10%;
}

Psuedo Classes

Use Psuedo Classes to active certain attributes when a certain action is preformed.

button:hover {
background-color: #772014;
color: #fff;
}
button:active {
font-size: 180%;
box-shadow: 0 0 10px #000 inset;
}
button:focus {
outline: none;
border-color: #8ac4ff;
}

Pseudo Elements

Combined with psuedo classes, psuedo elements can be used to style a specific part of the selected elements. Use the ::before or the ::after to apply content to the selected element before or after.

.all-boxes-span:hover::before {
content: "Class for the container of all snippet boxes.";
position: absolute;
border: 2px solid aqua;
width: 150px;
background-color: var(--tooltip-color);
top:15px;
}

Variables

Use variables in CSS to apply the smae attribute to multiple selectors. First use :root, then inside the :root name the variables with --variable-name. This snippet adds variables to this websites tooltip psuedo elements.

:root {
--tooltip-color: rgb(20, 20, 20, .85);
--tooltip-border: 2px solid aqua;
--tooltip-width: 150px;
--tooltip-top: 15px;
}
